# UART Transmitter & Receiver

## Overview

This project implements an 8-bit UART (Universal Asynchronous Receiver Transmitter) in Verilog HDL for serial communication between a PC and an FPGA.

The design was implemented and hardware validated on the Nexys A7 (Artix-7 100T) FPGA board using Xilinx Vivado.

## Features

- 8-bit UART Transmitter
- 8-bit UART Receiver
- Baud Rate: 9600 bps
- Start and Stop Bit Detection
- Baud Rate Generator
- Hardware Validation using Tera Term

## Hardware

- FPGA Board: Nexys A7 (Artix-7 100T)

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator
- Tera Term

## Results

The UART transmitter and receiver were successfully implemented and tested on FPGA hardware. Serial communication was verified using Tera Term at 9600 baud.
